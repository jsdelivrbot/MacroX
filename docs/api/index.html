MacroX API documentation version 0.0.1

MacroX is a library to help with Haxe macros

Getting Started With MacroX

For getting started with Haxe, take a look at our introduction, read through the Haxe Manual or look at these use cases for Haxe, including tutorials and popular libraries:

package macrox

AMetadata

AMetadata is an abstract to either MetaAccess or Metadata.

AType

Abstract type to either Type or ComplexType.

ClassTypeTools

Tools to deal with haxe.macro.ClassType

ExprTools

Tools to deal with haxe.macro.Expr

FieldTools

Tools to deal with haxe.macro.Field

HxDefines

Contains all Haxe compiler defines This file is generated with the output of haxe --help-defines by Script.hx using HxDefines-in.hx

HxMetas

Contains all Haxe compiler metas This file is generated with the output of haxe --help-metas by Script.hx using HxMetas-in.hx

Hxon

Hxon is a object notation like JSON but it can be validated to a typedef

Metas

Abstract to be extended for generate abstract enums width metas.

Tools

Collection of tools to deal with types and exprs in macro.

TypeParamTools

Tools to deal with haxe.macro.TypeParam

TypeTools

Tools to deal with AType
